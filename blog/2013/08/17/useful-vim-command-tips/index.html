Useful Vim Command Tips

Aug 17th, 2013

ctrl+w+(w, h, j, k,l)

ctrl+b,

ctrl+w,

gg  移到整份文件的最上方

G  移到整份文件的最下方

H  移动到当前屏幕的最上方

L  移动到当前屏幕的最下方

ds”  删除周围的双引号

cst  删除周围的tag

yss”  把整行加上双引号

:new  建立水平视窗

:vnew  建立垂直视窗
